# To-Do List Manager 📝

A simple console To-Do List application written in Python.

## Features

* Add tasks
* View task list
* Delete tasks
* Exit program
* Empty list checking

## Technologies

* Python 3

## How to run

1. Clone repository:

```bash
git clone https://github.com/Abdur-Rohmannn/to-do-list-manager.git
```

2. Open project folder:

```bash
cd to-do-list-manager
```

3. Run:

```bash
python main.py
```

## Example

```text
1. Add
2. Show list
3. Delete
4. Exit
```
